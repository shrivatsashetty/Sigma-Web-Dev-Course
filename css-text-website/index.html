text formatting demo

The text in this box uses various text formatting properties. The para itself starts with an text-indent. The heading uses text-shadow, text-transform, text-indent. The entire paragraph is justified, also the last line, which is justified to center. The para uses a combination of contrasting background-color and text color. The line spacing is set to 2.5, word spacng to 6px and the text spacing set to 3px. The default underline has been removed from the following link: google.com and a wavy line-through is added instead. Similarly we can also add underline (on hover) or an overline and also set the text-decoration-style.
