The "Hello World!!!" Program

The "Hello World!!!" program is the most basic, simplest program to write in any language

The printf() function is used to display text to the std output 

                        
    #include <stdio.h>

    int main() {
        printf("Hello World!!!");
        return 0;
    }

The Output for the Code will be:
Process Complete with error code 0
Hello World!!!

Pressing the F9 or Ctrl + Enter
will run the program in most of the online browsers
